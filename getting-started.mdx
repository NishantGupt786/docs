# Getting Started

Welcome to **Agnet**! This guide will walk you through everything you need to know to get started.

---

## Docker Setup

For the full experience, including the web UI and isolated environments, we recommend using Docker.  
You can download it from the [official Docker website](https://www.docker.com/products/docker-desktop).

---

## Python Virtual Environment

Before installing dependencies, it's a good practice to create a virtual environment.  
Create one by running:

<CodeBlock code="python3 -m venv venv" language="bash" />

Then, activate it:

<CodeBlock
  code={`# On Windows (Git Bash or CMD)
.\\venv\\Scripts\\activate

# On macOS/Linux
source venv/bin/activate`}
  language="bash"
/>

---

## Installation

Create a virtual environment and install **Laddr** (CLI, core, API):

<CodeBlock code="pip install laddr" language="bash" />

Or develop against this repository (editable):

<CodeBlock code="pip install -e lib/laddr" language="bash" />

---

## Create a New Project

<CodeBlock
  code={`laddr init my_agent_system
cd my_agent_system`}
  language="bash"
/>

The project includes `agents/`, `workers/`, a `Dockerfile`, `docker-compose.yml`, and a `main.py` runner.

---

## Set API Keys

To quickly get started, add the following env vars to a `.env` file in your project root:

<CodeBlock
  code={`# .env
GEMINI_API_KEY=your_gemini_api_key
SERPER_API_KEY=your_serper_api_key`}
  language="bash"
/>

> **Note:** One of our tool templates (web search) requires a Serper API key; Gemini is used for LLM integrations. Set both before running the stack.

---

## Run the Stack (Docker)

<CodeBlock code="laddr run dev -d" language="bash" />
<CodeBlock code="docker compose up -d" language="bash" />

The framework uses Docker under the hood, so both commands start the same local stack.

Open the dashboard at  
`http://localhost:5173`  
and the API at  
`http://localhost:8000`.

---

## Add an Agent and a Tool

<CodeBlock
  code={`laddr add agent researcher --role "Researcher" --goal "Find facts" --llm-model gemini-2.5-flash
laddr add tool web_search --agent researcher --description "Search the web"`}
  language="bash"
/>
