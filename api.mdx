## Jobs

### Get Job

<CodeBlock code={`GET /api/jobs/{job_id}`} language="http" />
Retrieve a specific job by ID.

### List Jobs

<CodeBlock code={`GET /api/jobs?limit=50&offset=0`} language="http" />
List all jobs with pagination support.

### Replay Job

<CodeBlock code={`POST /api/jobs/{job_id}/replay
Content-Type: application/json

{
  "reexecute": false
}`} language="http" />
Replay a specific job.

## Prompts (Preferred)

### Create Prompt

<CodeBlock code={`POST /api/prompts
Content-Type: application/json

{
  "prompt_name": "coordinator",
  "inputs": {"topic": "Latest AI agents"}
}`} language="http" />

### Get Prompt

<CodeBlock code={`GET /api/prompts/{prompt_id}`} language="http" />
Retrieve details of a specific prompt.

### List Prompts

<CodeBlock code={`GET /api/prompts?limit=50`} language="http" />
List all available prompts with pagination.
